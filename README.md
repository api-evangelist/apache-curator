# Apache Curator (apache-curator)
Apache Curator is a Java/JVM client library for Apache ZooKeeper that includes a high-level API framework and utilities to make using ZooKeeper much easier and more reliable. It provides recipes for common distributed coordination patterns.

## APIs

### Apache Curator
Curator provides a high-level Java API with fluent builders for ZooKeeper operations, along with pre-built recipes for leader election, distributed locks, barriers, caches, counters, and service discovery.
